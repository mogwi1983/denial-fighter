# Agent Handoff Template

Use this when ending a session, especially if work is incomplete.

```md
## Handoff: YYYY-MM-DD

### Goal

- [What the user asked for]

### Completed

- [Concrete shipped change]
- [Concrete shipped change]

### Changed Files

- `path/to/file` - [why it changed]

### Verification

- `npm run build` - [pass/fail/not run and why]
- [Other checks]

### Tracker

- Step moved to `in_progress`: [title/id or not updated]
- Step moved to `completed`: [title/id or not updated]
- Sync endpoint run: [yes/no/result]
- Current next action: [tracker value or proposed value]

### Open Questions

- [Question/blocker, or "None"]

### Recommended Next Action

- [The smallest useful next task]
```
